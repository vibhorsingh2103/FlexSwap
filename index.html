What We Offer

FlexSwap offers a simple and seamless decentralized token swapping experience. Here’s how you can get started:

Step-1

Set Exchange Rate

Set the exchange rate between Token A and Token B. This allows you to specify how much Token B you will receive for each Token A.

Step-2

Add Liquidity

Add liquidity to the DEX by providing both Token A and Token B. This ensures that there is enough liquidity for swaps to occur.

Step-3

Approve Tokens

Approve both Token A and Token B for use in the DApp. This ensures that the contract can interact with your tokens for swaps.

Step-4

Swap Tokens

Now you can swap Token A for Token B. Input the amount of Token A to swap, and the DApp will calculate the equivalent amount of Token B based on the set exchange rate.

Step-5

View Transaction Status

After completing a swap, check the status to ensure that the transaction was successful. You’ll receive Token B in your wallet.
